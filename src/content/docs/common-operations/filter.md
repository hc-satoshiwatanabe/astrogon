## フィルター・データ取得

すべてのBIページで共通の操作です。画面上部の条件を設定してデータを取得します。

### 基本操作

1. 画面上部の条件（期間・倉庫・取引先など）を設定
2. 「取得」ボタンを押すとデータが表示されます
3. 「やり直し」ボタンで条件をリセットできます
4. 条件を変えるたびに「取得」を押すことで最新データを取得できます

### 主なフィルター項目

| フィルター | 説明 |
|---|---|
| 取引日 | 期間（開始〜終了）または単日指定 |
| 年度 | 対象年度の指定 |
| 倉庫 | 倉庫の絞り込み |
| 取引先 | 取引先の絞り込み |
| 商品 | 商品の絞り込み |
| 担当者 | 担当者の絞り込み |
| 都道府県 | 地域の絞り込み |
| 仕入先 | 仕入先の絞り込み（仕入系ページ） |
| 表示件数 | ランキング表示件数の指定 |
