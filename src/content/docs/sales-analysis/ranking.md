## 売上ランキング分析

各分類軸での売上金額・数量をランキング形式の横棒グラフで表示します。上位の商品・取引先・地域・担当者を一覧で把握できます。

### 売上金額ランキング

| ページID | ページ名 |
|---|---|
| E11 | 商品別 売上金額ランキング |
| E12 | 取引先別 売上金額ランキング |
| E13 | 都道府県別 売上金額ランキング |
| E14 | 担当者別 売上金額ランキング |
| E15 | 容器容量別 売上金額ランキング |

### 主な機能

- 上位N件（表示件数可変）をランキング表示
- グラフホバーで詳細値を表示
- 表示件数をプルダウンで変更（10件・20件・50件等）
- CSVエクスポートでデータダウンロード
- グリッドビューでテーブル形式でも確認可能
